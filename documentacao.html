Documentação Técnica do Projeto

1. Diagrama de Casos de Uso

Mostra as principais funcionalidades acessíveis ao cliente no aplicativo, como inserir salário, consultar receitas, metas, gastos e despesas.

2. Modelo Lógico do Banco de Dados

Apresenta as tabelas do banco de dados e os relacionamentos entre usuários, contas, movimentações e avaliações mensais.

3. Diagrama Entidade-Relacionamento (DER)

Ilustra visualmente as entidades do sistema (como Usuário, Conta e Movimentação) e seus relacionamentos.

4. Fluxo de Funcionalidades

Representa o caminho do usuário desde o login até funcionalidades como adicionar receitas, criar metas e visualizar gráficos.
